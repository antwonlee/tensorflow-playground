# Tensor
Rank, shape and data types are 3 important characteristics which define a Tensor


## Rank

|Tensor|Rank|
|---|---|
|4|0|
|[1,2,3]|1|
|[[1,2], [3,4]]|2|
|[[[1], [3], [4]]]|3|

## Shape

|Tensor|Shape|
|---|---|
|4|[]|
|[1,2,3]|[3]|
|[[1,2,3], [4,5,6]]|[3,2]|
|[[[1], [2], [3], [4]]]|[2,2,1]|
